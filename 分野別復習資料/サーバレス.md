## サーバレス
- RDS Proxy
    - アプリケーションとRDSデータベース間の仲介役
    - 例
        - Lambda -> RDS Proxy -> RDS
- SQS
    - メッセージはデフォルトではデッドキューに移動しない
    - 可視性タイムアウト
    - メッセージ保存期間はデフォルト4日間、最長14日間
    - キューに優先度を設定することも可能（優先度高いものから処理）
- Lambda
    - 同時実行数：1000
    - 一時ボリューム量最大制限：512MB
- CloudFormation
    - Resourcesだけは必須
